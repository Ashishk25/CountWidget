# CountWidget
Android widget for select no of items. Usefull in situations like shopping cart , orders etc

## add this to your build.gradle app

```
dependencies {

	        implementation 'com.github.Ashishk25:CountWidget:1.1'
	}
```  
## or

```  
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
